Leetcode_As_C--1-两数之和

2019-11-08

描述

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/two-sum

暴力法

将所有组合方式都与目标进行等值判断。若共有n个数，则有Cn2C^2_nCn2​种组合方式。对于nums[0]，从nums[1]到nums[n-1]共n-1种组合。C代码如下：

int* twoSum(int* nums, int numsSize, int target, int* returnSize){
    int * result = (int *)malloc(sizeof(int) * 2); // 在内存中动态地分配一块大小为sizeof(int)*2的内存空间
    for (int i = 0; i < numsSize - 1; i++) {
        for (int j = i + 1; j < numsSize; j++) {
            if (nums[i] + nums[j] == target) {
                *returnSize = 2;
                result[0] = i;
                result[1] = j;
                return result;
            }
        }
    }
    return result; 
}

哈希表法

哈希表就是一种以 键-值(key-indexed) 存储数据的结构，我们只要输入待查找的值即key，即可查找到其对应的值。

https://www.cnblogs.com/yangecnu/p/Introduce-Hashtable.html

哈希表使用如下链表存储。

struct hash_data {
    int key;
    int data;
    struct hash_data * next;
};

链表放在下面的结构体中以便使用。

 struct hash_table {
    struct hash_data ** head; // 链表数组
    int hash_width;
};
